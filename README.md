# 자세 알리미 (Posture Alert)

앉은 자세를 실시간으로 감지해서, 자세가 틀어지면 노트북 화면에 바로 경보를 띄워주는 웹 기반 소프트웨어예요.

> 해커톤용 프로젝트예요. 별도 하드웨어 없이 **스마트폰 카메라 + 노트북 브라우저**만으로 동작하는 걸 목표로 해요.

## 문제의식

- 오래 앉아 일하면 거북목·구부정한 자세가 습관이 돼요.
- 정작 본인은 자세가 틀어진 걸 실시간으로 알아채기 어려워요.
- 값비싼 기기 없이, 이미 가진 폰과 노트북만으로 자세를 잡아주고 싶어요.

## 동작 방식

```
[스마트폰 카메라]  --(웹캠으로 연결)-->  [노트북 브라우저]
   옆모습 촬영                              │
                                            ├─ 영상 프레임 실시간 분석 (CV)
                                            ├─ 자세 점수 계산 (각도 기반)
                                            └─ 점수 낮으면 경보 표시
```

- 스마트폰을 옆에 세워두고 **옆모습(측면)** 을 비춰요.
- 폰을 노트북의 웹캠으로 연결해요.
  - 아이폰 + 맥: **연속성 카메라(Continuity Camera)**
  - 그 외: **Camo / DroidCam** 같은 앱
- 노트북 입장에선 그냥 일반 웹캠 하나로 잡혀요.
- 브라우저가 그 영상을 받아서 **전부 브라우저 안에서** 분석해요. (MVP는 백엔드 없음)

## 기술 스택

### 컴퓨터 비전 (핵심)

- **MediaPipe Pose Landmarker (BlazePose)**
  - 몸의 33개 관절 좌표를 실시간으로 뽑아줘요.
  - 브라우저에서 바로 돌아가요 (`@mediapipe/tasks-vision`).
  - 가볍고, 노트북에서 30fps 넉넉히 나와요. (1초 단위 분석은 아주 여유로움)
  - 관절마다 `visibility` 값이 있어서, 옆모습에서 안 보이는 반대쪽을 걸러내기 좋아요.

### 자세 점수 로직 (직접 구현)

CV는 관절 좌표만 줘요. "자세 점수"는 그 위에 얹는 **각도 계산**으로 만들어요.

- **거북목 (Forward Head)**
  - 귀 → 어깨 벡터와 수직선 사이 각도 (CVA, craniovertebral angle)
  - 고개가 앞으로 나올수록 각도가 나빠져요.
- **구부정함 (Slouch)**
  - 어깨 → 엉덩이 선의 수직 대비 기울기
  - 상체가 앞으로 무너질수록 나빠져요.
- 두 각도를 **0~100점**으로 환산해요.
- 몇 프레임 평균을 내서 떨림(jitter)을 없애요.
- 점수가 임계값 밑으로 내려가면 경보를 띄워요.

### 프론트엔드

- **웹 (브라우저 단독)** — HTML/CSS/JS 또는 가벼운 프레임워크
- 카메라 영상 위에 관절 스켈레톤을 겹쳐 보여줘요.
- 실시간 자세 점수를 화면에 표시해요.
- 자세가 틀어지면 시각 경보 (+ 선택적으로 소리)

## MVP 범위 (해커톤)

꼭 넣을 것만 넣어요.

- [ ] 폰을 웹캠으로 연결 → 브라우저에서 영상 받기
- [ ] MediaPipe로 관절 좌표 실시간 추출
- [ ] 거북목·구부정함 각도 계산 → 자세 점수(0~100)
- [ ] 화면에 실시간 점수 + 스켈레톤 표시
- [ ] 점수 낮을 때 경보 표시
- [ ] 기준 자세 캘리브레이션 (바른 자세 1회 등록)

### MVP 이후 (지금은 안 함)

- 자세 기록·통계 대시보드
- 알림 히스토리 / 리마인더
- 여러 사람·여러 각도 지원
- 데스크톱 앱 패키징

## 실행 방법

> 아직 구현 전이에요. 코드가 생기면 채울 예정이에요.

## 기술 선택 메모

- MoveNet(TF.js)도 후보였지만, 옆모습에서 3D 좌표를 주는 MediaPipe가 더 안정적이라 채택했어요.
- YOLO-Pose는 정확도는 높지만 백엔드가 필요해서 해커톤 MVP엔 과해요. 정확도가 부족하면 그때 검토해요.
